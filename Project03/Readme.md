# Tata Virtual Internship: Retail Strategy & Data Visualization

## 📌 Project Overview
As part of the Tata Group virtual internship via Forage, I functioned as a Data Analyst providing insights for the CEO and CMO. The goal was to analyze a dataset of over 1,36,000+  rows to identify expansion opportunities and revenue drivers.

## 🛠️ Technical Workflow
* **Data Cleaning:** Used Power Query to handle "data debt," fixing inconsistent records and Date-Locale errors.
* **Analysis:** Executed time-series analysis and regional segmentation to isolate high-performing markets.
* **Visualization:** Built a multi-layered Power BI dashboard focused on executive-level storytelling.

## 📈 Key Insights
* **Revenue Growth:** Managed a dataset representing **$10M+ in total revenue** and **5.1M+ units sold**.
* **Market Expansion:** Identified the **Netherlands and Germany** as the primary high-growth markets outside the UK.
* **Executive Impact:** Designed visuals that allow the CEO to pivot resources based on margin health and unit demand.
